## About

Application Security Engineer with hands-on experience across pentesting (web, API, mobile), vulnerability management at scale, security tooling administration, and DevSecOps pipeline design. I work close to engineering teams: triaging findings, tuning security tools, building pipeline gates, investigating incidents, and reviewing architecture to reduce real risk without slowing delivery.

I care about scope boundaries, accuracy over inflation, and turning findings into something developers can actually act on.
